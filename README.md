# Easy DOM Query Example

本仓库为 [Easy DOM Query](https://github.com/pushmetop/easy-dom-query) 的代码样例。

## 使用
 
### 克隆仓库

```
git clone git@github.com:pushmetop/easy-dom-query-example.git
```

### 安装依赖代码

```
cd easy-dom-query-example
yarn install
```

### 启动开发环境

```
cd easy-dom-query-example
yarn dev
```
